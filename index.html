hexo教程（一）——使用hexo+github搭建个人博客搭建个人博客

使用hexo+github搭建个人博客

安装node.js

安装git环境

由于之前在做项目的过程中已经安装过node.js和git，故此处不再演示，可以参考网上其他教程

安装hexo

想必到了这里，你已经安装好了node.js和git，下面开始安装hexo。

  • 在某一磁盘创建文件夹
    我选择的是G盘，文件夹名为 Blog

  • 进入Blog文件夹，右键鼠标->选择Git Bash Here

  • 输入 npm install -g hexo-cli ，耐心等待一段时间后，如下图所示

  • 输入 npm install hexo -save，耐心等待一段时间后，如下图所示

  • 我们发现，Blog文件夹中多了如下所示的内容

在接下来的步骤中，我出现了 Command Not Found的情况，在网上查找解决办法，于是有了后续步骤

  • 在Blog文件夹中新建文件夹，命名为hexo
  • 关闭当前Git Bash，进入hexo文件夹，右键鼠标->选择Git Bash Here。或者直接在当前的Git Bash中输入 cd hexo
  • 输入hexo init，初始化hexo环境，耐心等待一段时间后，如下图所示
  • 输入npm install，安装npm依赖包，耐心等待一段时间后，如下图所示
  • 输入hexo generate或者是hexo g，生成静态页面，耐心等待一段时间后，如下图所示
  • 输入hexo server或者是hexo s，生成本地服务，如下图所示。我们每次写完博客，也可以现在本地预览一下，然后再发布到网上去。
  • 接着，我们在浏览器中访问http://localhost:4000/，如下图所示。这就是在本地生成的一个博客。
  • 在Git Bash中按下Ctrl+C，关闭当前端口服务。
  • 到这里，我们的本地博客就已经搭建完成了。下一篇博客将介绍如何与Github连接，将博客上传至Internet
    【博主说】
    1. 第一次写博客，如有不当之处，请多多包涵
    2. 在上文中多次提到“耐心等待一段时间”的字样，不知道是因为我网络环境不佳，还是都有这个问题，等待的时间大多很长
